# Par Level AI — parlevelai.com

Managed Procurement for Healthcare Facilities. Powered by MeddMax LLC.

## Deployment

1. Connect this repo to Vercel at [vercel.com/new](https://vercel.com/new) — import `github.com/dmweissman/PARLEVELAI`
2. Replace `YOUR_FORM_ID` in `index.html` with your [Formspree](https://formspree.io) form ID
3. Add images to the `images/` folder: `desk-flatlay.png`, `ipad-dashboard.png`, `spend-chart.png`
